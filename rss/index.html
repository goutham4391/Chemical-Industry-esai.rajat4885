RSS Feed Delivery

Eisai Co., Ltd. publishes news releases and global site updates (Latest News, Information) using an XML-based contents delivery format known as RSS.

What is RSS?

RSS (Really Simple Syndication) is an XML-based format for sharing and distributing frequently updated Web content (news headlines, summaries, etc.). Text files and contents published in RSS format are also referred to as RSS feeds. Using an RSS reader, you can automatically find out what information has been updated on a given site.

What is an RSS Reader?

RSS readers are a type of software used to efficiently obtain and view information from websites with RSS feeds. By subscribing to and importing RSS feeds that list site updates and summaries, you can check the latest updates published on a given site.
